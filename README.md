# Masked Autoencoder (MAE) Image Reconstruction

This repository contains the deployment code for a Masked Autoencoder (MAE) model. The application provides an interactive web interface using Gradio to demonstrate how the MAE model reconstructs missing patches from an input image.

🚀 **[Try the Live Demo on Hugging Face Spaces!](https://huggingface.co/spaces/ShayanAmin/MAE-Demo)**
